# REQUISITOS FUNCIONAIS

## RF001 - Dashboard

O sistema deverá apresentar um painel com:

- Oportunidades novas
- Oportunidades em cotação
- Oportunidades vencidas
- Cotações pendentes
- Propostas enviadas

---

## RF002 - Oportunidades

O sistema deverá permitir:

- Importar oportunidade
- Cadastrar manualmente
- Pesquisar
- Filtrar
- Editar
- Excluir

---

## RF003 - Itens

Cada oportunidade poderá possuir vários itens.

Cada item deverá conter:

- Código
- Descrição
- Quantidade
- Fabricante
- Código fabricante
- Categoria
- Norma
- Dimensões

---

## RF004 - Fornecedores

Cadastrar:

- Nome
- Contato
- Email
- Telefone
- Fabricantes atendidos

---

## RF005 - Cotações

Permitir:

- Solicitar orçamento
- Registrar resposta
- Comparar preços
- Registrar prazo

---

## RF006 - Histórico

Armazenar:

- Preços anteriores
- Fornecedor
- Data
- Resultado da licitação

---

## RF007 - Inteligência Artificial

A IA deverá:

- Extrair informações dos editais
- Classificar oportunidades
- Sugerir fornecedores
- Pesquisar histórico
